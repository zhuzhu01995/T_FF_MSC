# 数据文件夹

## 说明

本GPU版本使用与CPU版本相同的数据文件。

## 数据文件位置

### 方法1: 使用原CPU版本的数据（推荐）

数据文件应位于：
```
../T_FF_MSC_python/data/
├── yale.mat
├── ORL.mat
├── COIL20MV.mat
└── yaleB.mat
```

代码中已配置为使用这个路径，无需额外操作。

### 方法2: 复制数据到本文件夹

如果您希望在本文件夹中保存数据文件：

1. 从原CPU版本复制数据文件到此处：
   ```bash
   cp ../T_FF_MSC_python/data/*.mat .
   ```

2. 修改代码中的数据路径：
   ```python
   # 从
   data = loadmat('../T_FF_MSC_python/data/yale.mat')
   
   # 改为
   data = loadmat('data/yale.mat')
   ```

## 数据集信息

| 数据集 | 文件名 | 样本数 | 视图数 | 类别数 | 文件大小 |
|--------|--------|--------|--------|--------|----------|
| Yale | yale.mat | 165 | 3 | 15 | ~5MB |
| ORL | ORL.mat | 400 | 3 | 40 | ~12MB |
| COIL-20 | COIL20MV.mat | 1440 | 3 | 20 | ~45MB |
| Extended YaleB | yaleB.mat | 2414 | 3 | 38 | ~150MB |

## 数据格式

每个.mat文件包含以下字段：
- `X1`: 第一个视图的数据矩阵 (d1 × N)
- `X2`: 第二个视图的数据矩阵 (d2 × N)  
- `X3`: 第三个视图的数据矩阵 (d3 × N)
- `gt`: 真实标签向量 (N × 1)

其中：
- d1, d2, d3 是各视图的特征维度
- N 是样本数量

## 注意事项

- **.gitignore已配置忽略.mat文件**，避免将大文件提交到git仓库
- 数据文件较大，建议使用方法1（引用原数据）
- 如需下载数据，请参考原CPU版本的数据来源
